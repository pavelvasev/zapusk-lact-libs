# git

Предназначение: загрузка указанного репозитория

## Параметры

* **repo** адрес репозитория
* **dir** каталог куда загрузить

## Команды

* **apply** - если каталога нет, клонирует репозиторий; если каталог есть - вызывает git pull.

## Пример

```
####### my-app
[git]
repo=ssh://myhost.com/app1
dir=/var/app1
```
